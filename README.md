# Teddies Hair Design Website

A luxury hair salon website for Teddies Hair Design in Cannock, UK.

## Features

- Modern, responsive design with teal and gold color scheme
- Multi-step booking system
- Interactive gallery with lightbox
- Service menu with expandable categories
- Testimonials carousel
- Contact form
- Mobile-friendly navigation

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Playfair Display, Montserrat)
- Font Awesome icons
